# nes-mapmaker

A browser-based NES graphics editor and map maker — the 2.0 line of
[nesprite](https://github.com/djessemann/nesprite).

This repo started from a verbatim copy of nesprite's app (at commit `c577e94`, 2026-07-12) and
grows it into a multi-screen map editor: draw NES sprites and background tiles, then arrange
tiles into whole game worlds — Zelda-style screen-by-screen adventures, Dragon Quest-style
overworlds, towns, and dungeons — with export formats a NES game can ingest.

The original nesprite repo is preserved as-is, for posterity and as the clean minimal 1.0.

See `PLAN.md` for the roadmap and `CLAUDE.md` for project conventions.
