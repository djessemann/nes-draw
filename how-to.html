How to use nesprite draw

Make NES graphics in your browser on the real NES palette, and export to 6502 assembly, PNG, or GIF. The toggle at the top left of the editor switches between sprite mode (moving characters, animated) and tile mode (backgrounds you arrange into scenes and multi-screen worlds). The two are separate workspaces and auto-save.

1. Size

In the size box, set the tiles wide/tall and press resize (2 × 2 = a 16×16 character).

2. Colors

Tap one of the 3 color slots, then a color in the NES master palette to assign it (keys 1–3). Color 0 is transparent — use the eraser.

3. Draw

  • pencil (B) — paint with the active slot.
  • eraser (E) — erase to transparent.
  • fill (G) — flood-fill.
  • lasso (L) — outline pixels, then drag to move them.

↔ flip/↕ flip mirror the frame, clear empties it, ↶ undo (Ctrl/⌘+Z), grid toggles the lines.

4. Animate

+ add / duplicate / delete frames, set fps, and ▶ play to preview.

5. Export

  • save / load — your project as .json (holds both modes). Save asks for an optional project name, which becomes part of the file name and is remembered for next time.
  • 6502 — .s file: sprite_palette + CHR tile data. The file documents its own format (byte layout, tile order, timing) and includes each frame's pixels as comments.
  • png — frames in a row, transparent background.
  • gif — animated at the current fps, transparent background.

1. Tile size

Defaults to 1 × 1 (8×8). For a metatile, change the size box and press resize.

2. Colors

4 paintable slots. Slot 0 (bg) is the drawn backdrop, not transparent. Tap a slot, then a color in the NES master palette (keys 0–3).

3. Draw a tile

Tap a tile in the tileset to edit it, then:

  • pencil (B) — paint with the active slot.
  • eraser (E) — erase to the backdrop (color 0).
  • fill (G) — flood-fill.
  • lasso (L) — outline pixels, then drag to move them.

↔ flip/↕ flip mirror the tile, clear, ↶ undo (Ctrl/⌘+Z), grid toggles the lines.

4. Tileset

+ tile / duplicate / delete manage the bank; tap a thumbnail to select.

5. Arrange

Switch draw → arrange above the canvas. The selected tile is the stamp: place lays it down, erase clears a cell, clear/↶ undo work on the grid. For a mirrored tile, duplicate it and flip the copy in draw.

The screen box sets how big one screen is (in tiles).

6. Build a world

Set the world box to more than 1 × 1 and your map becomes a grid of screens — a Zelda-style adventure world, or a Dragon Quest overworld, town, or dungeon. Then:

  • A screen / world toggle appears. screen is where you paint — one screen at a time.
  • Move between screens with the ◀ ▶ ▲ ▼ buttons under the canvas (or the arrow keys); the number shows which screen is open.
  • world zooms out to the whole map. Tap any screen to open it for painting. Dashed lines mark the open screen.
  • clear empties just the open screen; in world view it clears the whole map (it asks first).

7. Export

  • save / load — your project as .json (holds both modes). Save asks for an optional project name, which becomes part of the file name and is remembered for next time.
  • 6502 — .s file: bg_palette + CHR tile data. The file documents its own format and includes each tile's pixels as comments.
  • png — tiles in a row, on the backdrop.
  • map — a PNG of your arranged layout — the whole world, every screen.
